# Contributing to Denarixx Sentinel

## Development
- Node.js 20 recommended
- Install deps: `npm ci`
- Build: `npm run build`
- Ensure `dist/index.js` is updated (ncc bundle) before PRs

## Pull Requests
- Use conventional commit prefixes: feat/fix/chore/docs/refactor/test
- Include PR description sections: What / Why / How tested (recommended)
- Add tests when changing core logic

## Releases
Releases are managed by Release Please on merges to `main`.
